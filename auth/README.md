# ant.design.pro-boot
ant.design.pro 管理系统后端实现


## 说明
api入口，作为聚合接口数据使用



## 工程结构
### 包分类
####    package
        api     对外服务接口
        service 服务层
        dao     数据访问层
        model   been
        config  配置
        utils   工具类

####    resources
     mybatis           Mapper 映射文件
     spring            spring配置文件（事物）
     application.yml   spring集成配置文件
     logback.xml       日志配置文件
     mysql.properties  数据库配置文件
     
#### webapp
        后端集成资源管理系统查询
